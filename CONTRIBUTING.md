## ü§ù How to Contribute

1.  **Fork the Repository**: Start by forking the [repo](https://github.com/VIKAS9793/IndiaAI-IDP-Platform.git) to your GitHub account.
2.  **Clone Locally**: `git clone https://github.com/YOUR_USERNAME/IndiaAI-IDP-Platform.git`
3.  **Create a Branch**: `git checkout -b feature/amazing-feature`
4.  **Commit Changes**: `git commit -m 'Add some amazing feature'`
5.  **Push to Branch**: `git push origin feature/amazing-feature`
6.  **Open a Pull Request**: Submit a PR to the `main` branch.
